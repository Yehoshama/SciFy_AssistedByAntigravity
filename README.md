# Sci-Fi Assisted by Antigravity 🚀

Welcome to the repository for my upcoming Sci-Fi novel, temporarily titled **The Zenith Anomaly**! *(Please note: this is just a working title and will very likely change as the story evolves!)*

This repository is something incredibly special. Every character profile, piece of worldbuilding lore, chapter outline, and draft you see in these folders was brainstormed, structured, and co-written entirely using **[Antigravity](https://deepmind.google/)**, an advanced agentic AI coding assistant built by Google DeepMind.

## 🌟 Why Antigravity for Creative Writing?

Antigravity is designed as a powerful pair-programming and software engineering tool. But out of curiosity, I decided to test its limits by throwing a creative writing project at it. The results absolutely blew me away. 

Instead of just acting like a standard chatbot, Antigravity functions like a true partner:
*   **It manages files autonomously:** It created my folder structure (`/characters`, `/drafts`, `/chapters`, `/locations`) without me ever leaving the chat interface.
*   **It remembers everything:** It seamlessly tracked complex, global rules we invented (like the multi-tier Profession system) and applied them correctly across multiple draft files.
*   **It adapts instantly:** When I changed my mind about a plot point, it didn't just remember it; it proactively went into multiple `.md` files in my workspace and rewrote paragraphs to ensure absolute logical consistency across the narrative.
*   **It operates the terminal:** It even pulled everything together into a beautifully formatted PDF by silently running terminal scripts in the background!

If you want to see what we built, dive into the folders! 
The story follows Kaelen Vance, an Intelligence Officer in a fiercely atheistic, hyper-rational civilization that has cured disease, mapped the genome, and concluded that they are alone in a dead universe.
